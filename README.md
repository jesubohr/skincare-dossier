# Skincare Dossier 🧴

**A Minimalist CRM for Modern Cosmetologists.**

Skincare Dossier is a clean, professional, and minimalist SaaS application tailored for cosmetologists and medical-spa practitioners. It simplifies client management by transforming disorganized notes into a sleek, card-based treatment history.

## ✨ Core Features

- **Elegant Authentication:** A secure, uncluttered login experience.
- **Client Directory:** A searchable grid of "Client Cards" providing instant access to contact info and last treatment dates.
- **Detailed Treatment Timelines:** A vertical feed of historical cases including observations, specific treatments (e.g., Microneedling, Chemical Peels), and follow-up recommendations.
- **Streamlined "Add Case" Drawer:** A focused slide-out interface for logging new treatments without losing context of the client profile.

## 🎨 Visual Identity

- **Aesthetic:** Modern "Medical-Spa."
- **Palette:** Pearl White, Sage Green accents, and soft Blush Pink highlights.
- **Typography:** High-legibility Sans-serif.
- **UI Components:** Card-based layouts with soft rounded corners and subtle shadows.

## 🚀 Tech Stack

- **Frontend:** React/Next.js with Tailwind CSS.
- **Icons:** Lucide-React for minimalist line iconography.
- **State Management:** React Query for seamless data fetching.
- **Backend:** Node.js / Supabase (Authentication & Database).

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/jesubohr/skincare-dossier.git

# Install dependencies
pnpm install

# Run the development server
pnpm dev

```
